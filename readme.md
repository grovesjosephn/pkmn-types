## Install

```bash
$ npm install --global pkmn-types
```


## CLI

```
$ pkmn-types --help

  Usage
    $ pkmn-types [type1] [type2]


  Examples
    $ pkmn-types Water [Ice]
    {
      strengths: [ 'Water', 'Ice' ],
      weaknesses: [ 'Grass', 'Electric', 'Fighting', 'Rock' ],
      immunes: []
    }
    {
      immunes: [],
      weaknesses: [ 'Water', 'Grass', 'Dragon' ],
      strengths: [ 'Fire', 'Ground', 'Rock' ]
    }
```
